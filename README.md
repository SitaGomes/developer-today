# How to run this project

## First go to the backend

### Run the following command on your terminal

```
cd backend
```

### Install the dependencies

```
npm i
```

### Run the api

```
npm run dev
```

## Now let's move to the frontend

### Open a new terminal window and run the command

```
cd frontend
```

### Install the dependencies

```
npm i
```

### Run the frontend

```
npm run dev
```
